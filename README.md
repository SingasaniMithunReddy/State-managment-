# HireJob.ai Portal - Angular 20 Frontend

A frontend-only hiring portal built with Angular 20 standalone components, route-based pages, mock data, and localStorage-powered interactions.

## What is included

- Landing page with featured jobs and product messaging
- Jobs listing page with client-side filters
- Job detail page
- Saved jobs page
- Application tracker page
- Employer marketing page
- Demo dashboard
- Login and register demo screens
- No backend code

## Tech choices

- Angular 20 standalone APIs
- Angular Router for SPA navigation
- Signals and computed state
- Built-in control flow (`@if`, `@for`)
- localStorage for saved jobs and application tracking

## Run locally

```bash
npm install
npm start
```

Then open `http://localhost:4200`.

## Build

```bash
npm run build
```
